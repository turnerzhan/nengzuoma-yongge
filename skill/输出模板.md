# 对咨询者输出（简单版）

**底座** = 原版 yongge-restaurant（`references/yongge-restaurant-output-template.md`）  
**加两样** = 连麦账表 + 双图摘要  

语气：短句、直接、劝退优先；像连麦，别像机关公文。详见 `风格指引.md` + `corpus/06`。

---

## 必含块

1. **结论灯** 🔴 / 🟡 / 🟢 + 人话一句  
2. **为什么**（带数据：地图数、规则名）  
3. **连麦账表**（有租/人工/流水时 **必贴**）  
4. **双图摘要**（有地址时：高德 + 百度；一家失败要写）  
5. **业态** 当前 + 最多 2 备选  
6. **条件 / 劝退**  
7. **上街再确认**  
8. **类比** 0～1 个  
9. 免责一行  

块可以 **人话串起来**，不必死念 `### 1 2 3`，但 **内容不能缺**。

---

## 连麦账表（锁定 · 咨询者必看）

标题：`这家餐饮店能救吗？` / `这店能不能开？`  
单位：**元/天**

| 指标 | 现状（或你这想法） | 调整后 |
|------|--------------------|--------|
| 营业额/天 | | |
| 毛利率 | | |
| 毛利/天 | | |
| 房租/天 | | |
| 人工/天 | | |
| 水电/物业/天 | | |
| 固定成本/天 | | |
| 保本线/天 | | |
| **纯利/天** | | |

公式：房租/天=月租÷30；固定=房+人+水；毛利=营业额×毛利率；保本=固定÷毛利率；纯利=毛利−固定。

## 双图摘要（示例句式）

```
【地图】
高德：…
百度：…（或：IP校验失败，未交叉）
车程：…
同业/大牌/学校小区商场：…
停车/交通：…
说明：过店人数需现场蹲点，地图只给 POI 代理。
```

---

## 长度

讲透即可；有图有账有双图时内容可以密。用户只要一句：灯 + 致命原因。
