# 保本线计算器

> skill 调用的算账核心。输入 4–6 个数字，输出保本线 + 斩杀线 + 诊断。

---

## A. 核心公式

```python
# 输入
营业额_日 = ... # 元/日
食材成本_日 = ... # 元/日
房租_月 = ... # 元/月
人工_月 = ... # 元/月
水电杂费_月 = ... # 元/月（可选，默认 1500）

# 计算
毛利率 = (营业额_日 - 食材成本_日) / 营业额_日
固定支出_月 = 房租_月 + 人工_月 + 水电杂费_月
保本线_月 = 固定支出_月 / 毛利率
保本线_日 = 保本线_月 / 30

# 判断
保本达成率 = 营业额_日 / 保本线_日
if 保本达成率 < 0.6:
 状态 = "[红] 斩杀线以下（做不了）"
elif 保本达成率 < 0.8:
 状态 = "[橙] 低于保本（持续亏损）"
elif 保本达成率 < 1.0:
 状态 = "[黄] 接近保本（小亏）"
elif 保本达成率 < 1.2:
 状态 = "[绿] 小赚（健康）"
else:
 状态 = "[绿绿] 盈利充裕"
```

## D. 投资回本周期

```python
月净利 = (营业额_日 × 30) × 净利率
回本周期_月 = 总投资 / 月净利

if 回本周期 > 24:
 评级 = "[红] 高风险（> 24 个月）"
elif 回本周期 > 18:
 评级 = "[黄] 偏慢（18–24 个月）"
elif 回本周期 >= 12:
 评级 = "[绿] 健康（12–18 个月）"
elif 回本周期 >= 6:
 评级 = "[绿绿] 优秀（6–12 个月，罕见）"
else:
 评级 = " 异常（< 6 个月，检查输入是否造假 / 是否快招话术）"
```

## F. 完整计算示例

### 输入
- 品类：奶茶
- 日营业额：800 元
- 食材一天用：250 元
- 月房租：8000 元
- 人工：2 个员工 × 4500 元 = 9000 元
- 水电杂费：1500 元
- 总投资：35 万（含 10 万贷款）

### 计算过程

```
毛利率 = (800 - 250) / 800 = 68.75%
固定支出 = 8000 + 9000 + 1500 = 18,500 元/月
保本线_月 = 18,500 / 68.75% = 26,909 元
保本线_日 = 26,909 / 30 = 897 元

保本达成率 = 800 / 897 = 89.2%
→ [黄] 接近保本（小亏）

月净利估算（按当前水平）：
 月营业额 = 800 × 30 = 24,000
 月毛利 = 24,000 × 68.75% = 16,500
 月净利 = 16,500 - 18,500 = -2,000（月亏 2000）

如果冲到 1200/日：
 月营业额 = 36,000
 月毛利 = 24,750
 月净利 = 6,250（月净利 6250）
 年净利 ≈ 7.5 万
 回本周期 = 35 / 7.5 × 12 ≈ 56 个月 [红] 过长
```

### 输出

```
---
 保本线诊断
---
你的毛利率：68.75%（饮品偏低，行业 70–80%）
月固定支出：18,500 元
月保本线：26,909 元
日保本线：897 元

你现在日均：800 元
保本达成率：89.2% → [黄] 接近保本，每月小亏 2000

即便你冲到 1200/日：
 月净利：6,250 元
 回本周期：≈ 56 个月 [红]

---
 勇哥结论
---
兄弟你这店，不是经营问题，是**模型问题**。
35 万投资 + 月净利撑死 6000 = 5 年回本。
还没算 10 万贷款利息。还没算你自己的工资。

现在两条路：
 A. 一个月内想办法冲到 1500+ / 日 → 继续观察
 B. 做不到就立刻转让止损 → 剩多少拿多少
```
