# 盈利与算账模型（勇哥口径）

> 能做吗？勇哥 **必须会算数**。地图解决「位置事实」，算账解决「账是否盖得住」。

可执行脚本（优先跑，别手算口胡）：

| 脚本 | 用途 |
|------|------|
| `tools/breakeven.py run` | 有日流水：保本线 / 达成率 / 月净利 / 回本；含 `lianmai_board` |
| `tools/breakeven.py plan` | 开店前反推日流水门槛 |
| `tools/breakeven.py iron` | 房租铁律 |
| `tools/quack_score.py` | 加盟快招风险分 |

> 注：旧 `scripts/profit_model.py` 已并入 `tools/breakeven.py`（ponytail 合并，公式未砍）。

---

## 1. 核心公式

### 1.1 毛利率

```
毛利率 = (日营业额 - 日食材成本) / 日营业额
```

未知食材时，用品类默认：

| 品类 | 默认毛利率 |
|------|------------|
| 快餐 | 52% |
| 小吃 / 早餐 | 55% |
| 正餐 | 60% |
| 烧烤 | 60% |
| 火锅 / 烘焙 | 65% |
| 奶茶 / 饮品 | 72% |
| 咖啡 | 75% |

### 1.2 保本线（斩杀线）

```
月固定支出 = 房租 + 人工 + 水电杂项(默认 1500)
月保本营业额 = 月固定支出 ÷ 毛利率
日保本线 = 月保本 ÷ 30
达成率 = 实际(或预估)日流水 ÷ 日保本线
```

| 达成率 | 状态 |
|--------|------|
| < 60% | 🔴 韭菜斩杀线 — 做不了 |
| 60%–80% | 🟠 持续亏损 — 限期冲或关 |
| 80%–100% | 🟡 接近保本 |
| 100%–120% | 🟢 健康小赚 |
| > 120% | 🟢 充裕 |

### 1.3 房租铁律

```
合理日营业额 ≈ 月房租 × 0.3
合理月营业额 ≈ 月房租 × 9
```

例：月租 1 万 → 日流水应在 **3000** 左右。长期明显低于此 = 租贵或选址错。

### 1.4 回本压力（开店前）

```
目标月净利 = 总投入 ÷ 目标回本月数(默认 18)
需要的日流水 = (目标月净利 + 月固定) ÷ (30 × 毛利率)
```

**开店前门槛日流水** = max(日保本, 房租铁律目标, 回本所需日流水)

冲不到这个数 → 模型先红，别谈梦想。

### 1.5 转让费三标准（提到转让时）

1. 约 3～4 个月净利润能赚回  
2. 转让费 ≤ 月租 × 15  
3. 接手首月营收 ≥ 转让费 ÷ 2

## 2. 什么时候必须算

| 用户给了 | 动作 |
|----------|------|
| 月租 | 至少 `iron` + 追问人工/投入 |
| 月租 + 人工 + 品类 | 跑 `plan` |
| 再加日流水/预估流水 | 跑 `run` / `breakeven` |
| 加盟费/总投入 | `plan` + `quack_score` |
| 什么数都没有 | 位置可先诊，但结论必须写「缺账，置信度降」并追问 |

**有数不算 = 执行失败。** 勇哥是算账的人，本 Skill 不能只靠嘴炮选址。

## 3. 命令速查

```bash
# 开店前反推
python scripts/profit_model.py plan \
  --rent 12000 --labor 9000 --investment 600000 --category 汉堡

# 有流水体检
python scripts/profit_model.py run \
  --daily-revenue 800 --daily-food-cost 250 \
  --rent 8000 --labor 9000 --investment 350000 --category 奶茶

# 只算房租铁律
python scripts/profit_model.py iron --rent 10000

# 加盟快招分
python scripts/quack_score.py \
  --source "抖音广告" --hq-city "台州" \
  --total-fee 600000 --direct-stores 1 --years 0.5 \
  --promises "全国首家,6个月回本"
```

无 Python 环境时：按上文公式手算，数字保留到个位即可，并在报告里写「手算」。

## 4. 输出里怎么展示（给用户看）

```text
【算账】
毛利率：xx%
月固定：x 元
日保本：x 元
房租铁律日目标：x 元
（若有投入）18 个月回本日目标：x 元
→ 门槛日流水：x 元
你现在/预估：x 元 → 达成率 xx% [灯]
```

先数字，再红黄绿，再嘴。
