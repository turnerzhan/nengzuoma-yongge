# 勇哥决策核（验铺精简版）

> 蒸馏自公开整理的勇哥餐饮选址/诊断方法论。仅供本 Skill 判断，不构成投资建议。

## A. 总原则

1. **选址决定餐饮成败约八成** —— 味道救不了烂位置。
2. **先算账、看红线、验商圈** —— 再谈梦想。
3. **能做的不保证赚；做不了的，别硬开。**
4. 结论只有三档：🔴 做不了 / 🟡 谨慎可谈 / 🟢 匹配（仍要蹲点）。

## B. 一票否决（触一条默认 🔴）

| # | 红线 | 说明 |
|---|------|------|
| 1 | 二楼及以上做杂牌餐饮 | 培训类除外；综合体高层同理 |
| 2 | 同品类大牌贴脸 + 无差异化 | 蜜雪旁杂牌奶茶 = 自杀式（脸盆姐模型） |
| 3 | 抵押房产 / 借亲友钱 / 贷款创业 | 资金来源红线；用户自报即否决 |
| 4 | 无销售推动 + 同业极稀 | 先烈区：客流没养起来 |
| 5 | 死街迹象强 | 转让招牌扎堆（图中大量转让）或用户自报 1/3 转让 |
| 6 | 房租铁律严重击穿 | 有月租时：合理日营收 ≈ 月租×0.3；差太远且业态救不了 |

用户未自报资金来源时，**不要假设**其借贷；可在建议里提醒「别抵押房产开店」。

## C. 四大选址模型（判断顺序）

```
匹配商圈 → 聚客街道 → 优质铺位 → 投入产出
```

### C.1 匹配商圈

- 客单价要贴商圈消费带；城中村硬上高客单、CBD 硬上 8 元粥 = 错配。
- 经验：客单价 ÷ 商圈人均消费 ≈ 0.6–1.2 合理（无数据时用「学校/社区/写字楼/景区」粗判）。

### C.2 聚客街道

- **有主题的街**（美食街、学校外街、早夜市）优于「随便几家底商凑的街」。
- 核心：晚饭点/周末，人是 **奔着这条街来** 还是 **路过**。

### C.3 优质铺位

- 金角银边草肚皮：角铺 > 边铺 > 中段。
- 一楼临街、门头可见、易接近（少台阶、少遮挡）。
- 门头宜宽；棺材铺（一楼极窄）差。

### C.4 投入产出

- 月房租 ≤ 预估月营收 × 15% 左右（铁律近似：1 万房租 → 日营收约 3000）。
- 12–18 个月回本预期；转让费另套三标准（见下）。

## D. 房租营业额铁律

```
合理日营业额 ≈ 月房租 × 0.3
合理月营业额 ≈ 月房租 × 9
```

预估或现状长期低于此 → 房租太贵或选址错。

## E. 同业浓度（用地图 500m 近似）

| 同品类数量（约） | 判定 |
|------------------|------|
| 很少（如 <10～15） | 偏红/黄：商圈未成熟或根本没需求 |
| 中等偏多（约 40～120 量级，商圈尺度） | 健康带：有流量也有缝 |
| 爆炸多 | 红海内卷，慎入 |

> 注意：原方法论的 40–120 常是「商圈尺度」；本 Skill 用 **500m POI 计数** 近似，需在话术中说「半径 500m 抽样」，避免装作全市精确普查。

## F. 销售推动业态（叠加）

500m 内统计是否存在：

- 学校  
- 写字楼 / 办公聚集  
- 大型小区  
- 商场主力店  

| 叠加 | 判定 |
|------|------|
| ≥2 | 绿向：客流源多元 |
| 1 | 黄：单点依赖（寒暑假/周末风险） |
| 0 | 红向：无来源 |

## G. 红灯位置（精简）

- 商住综合体「看起来很洋气」但无主动线  
- 新开发区 / 新楼盘养铺期  
- 二楼+ 餐饮  
- 大牌同品类贴脸开杂牌  
- 门前快车道、进不来  
- ≥3 级台阶（每级大幅砍进店）  
- 门口重遮挡  
- 周边转让扎堆  
- 十年租不出去的铺  

## H. 绿灯位置（精简）

- 封闭式学校门口（垄断客流，「关起来杀」）  
- 成熟社区底商 + 刚需品类  
- 已成型美食街 / 小吃街  
- 写字楼周边工作日简餐  
- 学校+写字楼+小区三角叠加  
- 阳街 + 角铺 + 一楼  

## I. 转让费三标准（用户提到转让费时）

1. 约 3～4 个月净利润能赚回  
2. 转让费 ≤ 月租 × 15  
3. 接手首月营收 ≥ 转让费 ÷ 2  

私下转让、二房东双重盘剥、带「品牌技术溢价」甩包 → 默认劝退。

## K. 综合裁决

```
IF 任一票否决: 🔴
ELIF 位置关键项多红 或 业态矩阵红: 🔴
ELIF 可改条件（砍租/换业态/强差异化）可救: 🟡
ELIF 位置与业态双过线 且 无否决: 🟢
ELSE: 🟡
```

🟢 必须附带：工作日+周末蹲点；双证与烟道排水电排查。  
🔴 直接说清「做不了」卡在哪条，不要「再观察观察」糊弄。
