# 街景 / 路段自动评分（验铺子集）

完整 12 项里，本 Skill 只对 **可看图或可地图** 的项打分；其余进「现场确认」。

## 分值约定

- 绿 = 10 · 黄 = 5 · 红 = 0  
- 看不清 = 不计分，记入 `needs_field_check`，并拉低置信度  

## 自动 / 半自动项

### 1. 楼层与动线（权重 1.5）· 图 + 商场 POI

| 情况 | 灯 |
|------|-----|
| 一楼临街，正门朝路 | 绿 |
| 一楼但巷内/侧门 | 黄 |
| 二楼+ / 综合体内部深处 / 整栋杂牌 | 红 |

### 2. 竞品贴脸（权重 1.5）· 高德 100m + 图内 logo

| 情况 | 灯 |
|------|-----|
| 无同品类大牌 | 绿 |
| 有大牌但用户业态差异化清晰 | 黄 |
| 同品类大牌贴脸 + 杂牌/无差异 | 红 |

### 3. 同业浓度（权重 1.5）· 高德 500m

| 情况 | 灯 |
|------|-----|
| 合理密集（有店有缝，非荒漠非炸街） | 绿 |
| 偏少或偏卷 | 黄 |
| 荒漠先烈 或 极端红海 | 红 |

用相对判断 + 给出具体计数；话术带「500m 抽样」。

### 4. 销售推动叠加（权重 1.0）· 高德

| 叠加数 | 灯 |
|--------|-----|
| ≥2 | 绿 |
| 1 | 黄 |
| 0 | 红 |

### 5. 台阶（权重 1.0）· 图

| 台阶 | 灯 |
|------|-----|
| 0～1 | 绿 |
| 2 | 黄 |
| ≥3 | 红（强调进店损耗） |

### 6. 门前遮挡（权重 1.0）· 图

| 情况 | 灯 |
|------|-----|
| 无 | 绿 |
| 轻遮挡 | 黄 |
| 紧贴重遮挡 | 红 |

### 7. 门头宽度 / 识别（权重 1.0）· 图

| 情况 | 灯 |
|------|-----|
| 宽敞、品类一眼懂 | 绿 |
| 偏窄或品牌字压过品类 | 黄 |
| 极窄棺材铺 / 满屏概念文案 | 红 |

### 8. 商圈类型（权重 1.5）· 图 + 高德

| 情况 | 灯 |
|------|-----|
| 美食街/学校街/成熟社区等聚客 | 绿 |
| 普通底商、单一写字楼 | 黄 |
| 综合体冷区、新开发区主干道、纯车流公路 | 红 |

## 不自动（必列现场确认）

- 精确转让率  
- 双证、烟道、隔油池、商业电  
- 阴阳街精确朝向（可猜但标不确定）  
- 真实过店人数（必须蹲点）  

## 汇总

```
加权粗分 = Σ(分 × 权重) / Σ(已评分项权重) × 10
```

| 粗分倾向 | 档 |
|----------|-----|
| 高且无否决 | 可冲 🟢（仍看业态） |
| 中 | 🟡 |
| 低或多项 0 | 🔴 |

**一票否决优先于分数。** 业态匹配见 `category-fit-matrix.md`，与位置分取严。
