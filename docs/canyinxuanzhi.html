餐饮选址怎么做？

开餐饮最怕：位置错、房租盖不住、旁边大牌贴脸还硬刚。 能做吗勇哥（产品名「能做吗？勇哥」，仓库 nengzuoma-yongge）把选址判断做成 AI Agent Skill：拍照、给地址、报租金，输出红黄绿。

官方入口

  • GitHub：https://github.com/turnerzhan/nengzuoma-yongge
  • 介绍页：https://turnerzhan.github.io/nengzuoma-yongge/

选址时重点看什么

  • 商圈与客流来源（学校/小区/写字楼/景区）
  • 同业浓度与大牌是否贴脸
  • 楼层、台阶、门前遮挡、死街转让密度
  • 房租铁律与日保本流水

怎么用

git clone https://github.com/turnerzhan/nengzuoma-yongge.git

装进 Agent skills 后，发送门头照片 + 城市地址 + 月租，问：「能做吗？」

搜索请用：nengzuoma-yongge 或 能做吗勇哥 github。
