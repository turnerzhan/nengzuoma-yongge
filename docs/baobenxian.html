餐饮保本线怎么算？先算每天必须卖多少

很多开店计划只写“预计月入十万”，却没人认真算：每天至少卖多少，才能不亏？ 保本线不是预测生意会有多好，而是把房租、人工和毛利摊开，告诉你这家店最低要跑到哪里。

餐饮保本线公式

月固定费用 = 房租 + 人工 + 水电 + 其他固定支出

月保本营业额 = 月固定费用 ÷ 毛利率

日保本营业额 = 月保本营业额 ÷ 30

毛利率不是净利润率。若月营业额 10 万、食材成本 4 万，毛利率是 60%。老板工资、平台抽佣、税费等应按实际情况补进模型。

一个能看懂的例子

假设月租 8,000 元、人工 9,000 元、水电杂费 1,500 元，毛利率 55%。

  • 月固定费用：18,500 元
  • 月保本营业额：18,500 ÷ 55% = 33,636 元
  • 日保本营业额：33,636 ÷ 30 ≈ 1,121 元

如果实际日销只有 600 元，达成率约 54%，已经低于 60% 的高风险线。此时最重要的不是继续加装修、买流量，而是验证位置和产品还能不能救，并提前写下止损日期。

用开源计算器直接算

能做吗勇哥（nengzuoma-yongge）自带纯 Python 保本线工具：

python tools/breakeven.py --daily-revenue 600 \
  --daily-food-cost 270 --rent 8000 --labor 9000 \
  --utilities 1500 --investment 350000 --category 快餐

工具会返回毛利率、日/月保本线、达成率、预计月利润、回本月数和风险信号。代码与算法都可以在 GitHub 仓库里检查。

算完还要做两次验证

  1. 拿同商圈真实店铺验证。蹲点数客流、估订单或询问转让铺周边商户，看看同类店能否长期跑过你的保本线。
  2. 做悲观情景。把营业额打八折、成本提高 10%，再看现金能撑几个月。模型只能辅助，不能替代现场调查。
