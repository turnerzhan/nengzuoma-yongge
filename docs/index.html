这是什么

原创 Agent Skill：帮小生意在掏钱前做门头验铺、选址诊断、加盟风险与保本线计算。支持 Claude / Cursor / Grok 等可加载 Skill 的 Agent。

核心能力

  • 门头 / 街景红黄绿诊断
  • 高德主 + 百度备 POI 商圈供血
  • 保本线 / 房租铁律 / 回本压力
  • 加盟快招评分与案例库类比
