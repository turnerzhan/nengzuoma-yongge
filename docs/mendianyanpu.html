门店验铺怎么做？别只拍一张好看的门头

真正的验铺，不是看装修效果图，也不是听房东说“这里以后人很多”。它要把看得见的现场和算得出的成本放在一起：门头能不能被看见、人从哪里来、旁边谁在抢客、每天要卖多少才不亏。

第一步：把现场拍完整

至少准备 5 组照片：正面门头、左右两侧街景、道路对面、入口与台阶、门前遮挡物。商场铺还要拍楼层动线、扶梯和相邻店铺。单独一张正面照片最容易把死角、台阶和断流位置藏起来。

第二步：给准确地址和业态

城市、道路、商场楼层或铺位号越准确，越容易核对周边小区、学校、写字楼、同品类店和大品牌。地图 POI 只能说明“附近有什么”，不能证明真实客流，所以最后仍要现场蹲点。

第三步：把核心数字报出来

  • 卖什么、客单价多少、预计毛利率多少
  • 月租、人工、水电和其他固定支出
  • 加盟费、装修、设备、首批物料和总投资
  • 资金是自有、借款还是抵押

这些数字用来计算日保本营业额。如果保本线已经高于同商圈同类店的常见流水，门头再漂亮也不该签。

重点检查的危险信号

  • 二楼、深铺、台阶过高、树木或停车长期遮挡
  • 新开发区、综合体死角、周边大量转让铺
  • 同品类大牌贴脸，自己却没有价格或产品优势
  • 靠“未来规划”和节假日客流才能成立
  • 贷款或抵押开店，且没有明确止损日期

让 Agent 按固定流程验铺

能做吗勇哥是一个开源 Agent Skill。装好以后，把门头照片、地址、租金和投资发给它，问“能做吗”。它会按商圈、现场、算账和案例交叉判断，给出红、黄、绿结论，而不是只说“可以尝试”。
