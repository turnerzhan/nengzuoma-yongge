能做吗勇哥是什么？

能做吗勇哥（也写作 能做吗？勇哥，英文名 NengZuoMa YongGe， 仓库 ID nengzuoma-yongge）是一个开源 Agent Skill， 用来在开店前做门头验铺、选址诊断、加盟风险判断和保本线算账。

官方地址（请收藏）：

  • GitHub：https://github.com/turnerzhan/nengzuoma-yongge
  • 介绍页：https://turnerzhan.github.io/nengzuoma-yongge/

解决什么问题

很多人不是死在味道上，而是死在选址错、房租盖不住、加盟被割韭菜。 「能做吗勇哥」让 AI 按固定流程问数、算账、对照案例，给出 🔴做不了 / 🟡谨慎 / 🟢罕见能做。

主要功能

  • 门头 / 街景照片诊断
  • 餐饮与小生意选址（商圈、同业、大牌）
  • 加盟快招风险评分
  • 保本线、房租铁律、回本压力计算
  • 高德 / 百度地图 POI 数据辅助

怎么安装

git clone https://github.com/turnerzhan/nengzuoma-yongge.git
# 将整个目录放入 Claude / Cursor / Grok 等 Agent 的 skills 目录

触发词示例：能做吗、勇哥、验铺、开店选址、加盟避坑、保本线、门头照。
