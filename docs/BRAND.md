## 2. 声音：怎么说话才有流量

### 要

- 短句。三行内给结论。  
- 先骂坑，再给数据。  
- 夸张但可核对（地图数字、投入数字得真）。  
- 金句收尾只留一句。  

### 不要

- 「或许、可能、仁者见仁」  
- 「加油你一定行」  
- 长篇行业黑话堆砌  
- 替加盟商圆场  

### 语气标杆

| 场景 | 开口 |
|------|------|
| 红灯 | 「做不了。今天就停。」 |
| 黄灯 | 「能谈，但先满足这些，否则一样死。」 |
| 绿灯 | 「罕见。还是先蹲点，别飘。」 |
| 加盟 | 「全国首家三个字出现，你就已经是样品。」 |

## 3. 传播钩子（可直接做标题）

1. 拍一张门头，3 分钟骂醒你  
2. 勇哥说做不了：这座铺别签  
3. 全国首家汉堡？先把 60 万放下  
4. 餐饮创业反诈：验铺比配方重要  
5. 高德扫完街，勇哥判死刑  
6. 别用全家积蓄给房东送钱  
7. 红灯一票否决，没有「再观察观察」  
8. 能做吗？勇哥。不能做的别硬开  

### 视频结构（15–45 秒）

```text
0-3s   门头特写 + 大字「能做吗？」
3-8s   甩地址 / 加盟费数字
8-25s  地图结论 + 红灯理由 3 条
25-35s 「勇哥说做不了」
35-45s CTA：评论区丢你的门头
```

## 4. 视觉资产

| 文件 | 用途 |
|------|------|
| `logo.jpg` | 砸章冲击，头像 / 角标 |
| `banner.jpg` | 仓库封面、横版封面 |
| `viral-uncle.jpg` | 话题向封面、怒指镜头 |
| `viral-x.jpg` | 「禁止开店」情绪图 |
| `poster-hero.html` | 横版文字海报（截图传播） |
| `poster-square.html` | 方图海报（抖音/小红书） |

### 色感

- 印泥红 `#B91C1C` / `#DC2626`  
- 警戒黄 `#FACC15`  
- 死黑 `#000` / `#0A0A0A`  
- 少用「温和蓝、互联网紫」——那是 SaaS，不是行刑队  

### 风格关键词

夸张 · 印章 · 警戒线 · 漫画冲击 · 高对比 · 少灰字
