# 05 · 品类与赛道

> 选品类比选铺位更重要。错的品类，再好的位置也救不了；对的品类，普通位置也能活。

---

## A. 中年破产四件套（核心红海清单）

```
 奶茶 · 汉堡 · 咖啡 · 烘焙
```

四件套共同特征：
- 看起来门槛低（"我家也做过""我朋友会做"）
- 实际竞争极度红海，巨头林立
- 单店模型已被规模玩家压到极致
- 小白几乎无法存活

### A.1 为什么这四个最容易让中年破产

| 品类 | 巨头 | 价格战现状 | 小白机会 |
|---|---|---|---|
| 奶茶 | 蜜雪、霸王茶姬、古茗、茶百道、沪上 | 4 元起步 | 几乎为零 |
| 汉堡 | 塔斯汀、华莱士、肯德基、麦当劳 | 8.9 元一个 | 几乎为零 |
| 咖啡 | 瑞幸、库迪、Manner | 9.9 元 → 6.9 元 | 几乎为零 |
| 烘焙 | 各大连锁 + 烘焙工坊 | 高损耗 + 高频次 | 极低 |

### A.2 为什么偏偏是中年人栽倒？
- 有积蓄、有焦虑、有时间，是最优质的"韭菜画像"
- 自我认知偏高，觉得"我做过销售/管理/工程，做生意不在话下"
- 对短视频 / 加盟广告辨识能力低
- 抗风险能力差但赌博心态重（"我必须翻身"）

> **「县城中产破产四件套」与「中年破产四件套」内容相同，只是发生地从一二线下沉到县城。**

## B. 勇哥推荐的低风险刚需品类

```
 猪脚饭 / 卤肉饭 / 快餐盒饭
 早餐铺 / 包子铺 / 豆浆油条
 面馆 / 米线 / 馄饨 / 砂锅
 葱油饼 / 煎饼果子 / 鸡蛋灌饼
 烤面筋 / 鸡叉骨 / 摆摊小吃
 社区便餐 / 菜市场熟食
```

**共同特征**：
- 客单价低（10–25 元为主）
- 复购率高（一周来 3–5 次正常）
- 食材标准化（不依赖"大厨"）
- 人工少（1–3 人能撑起来）
- 投资小（5–15 万启动）

## C. 三类生意模式分类

> 勇哥独家三分法（强烈推荐进 skill）：

```
小吃 = 流量生意 → 靠位置 + 路过即买
刚需快餐 = 复购生意 → 靠味道 + 价格稳定
早餐 = 顺路生意 → 靠通勤路径 + 出餐速度
```

不同模式，选址逻辑、定价策略、运营重点都不一样：

| 模式 | 关键变量 | 选址首选 | 经营重点 |
|---|---|---|---|
| 流量生意（小吃） | 自然路过 | 商业街 / 夜市 / 学校外 | 引人 + 出品快 |
| 复购生意（快餐） | 老客 | 社区 / 写字楼 / 工厂 | 稳定味道 + 性价比 |
| 顺路生意（早餐） | 通勤路径 | 地铁口 / 公交站 / 园区 | 出餐快 + 干净 |

## D. 选品五原则

1. **做刚需不做非刚需**
- 人每天都要吃饭，但不是每天都要喝奶茶
2. **做大众价格带不做高端**
- 消费降级趋势下高端餐饮萎缩
3. **做标准化不做依赖厨师**
- 厨师一走店就倒
4. **资金有限：做自营不做加盟**
- 加盟费 + 物料绑定吃掉绝大多数利润
5. **敢为人后而非敢为人先**
- 成熟品类 > 创新概念

## E. 高风险品类扩展清单（破产四件套+）

除了核心四件套，以下也属于高风险：

| 品类 | 风险点 |
|---|---|
| 主题餐厅（女仆 / 二次元 / 太空 / 监狱） | 噱头大于实质，难持续 |
| 网红概念店 | 周期短，热度过去就死 |
| 大店正餐（200㎡+） | 重资产，回本周期长 |
| 夜宵烧烤（无固定客群的） | 季节波动大，城管风险 |
| 高客单价火锅 | 消费降级首当其冲 |
| 中央厨房 / 半成品店 | 模型未跑通就被资本玩死 |
| 健康减脂餐 | 客群窄，复购难 |
| 玻尿酸 / 胶原蛋白等"概念饮品" | 几乎全是快招 |

## F. 县城 vs 一二线品类策略差异

| 维度 | 县城 / 三四线 | 一二线 |
|---|---|---|
| 主流客单 | 8–18 元 | 20–40 元 |
| 受欢迎品类 | 米粉、煎饼、卤肉饭、平价火锅 | 精致快餐、咖啡、轻食 |
| 不受欢迎品类 | 高端咖啡、网红轻食、健身餐 | 浓油重盐大盘菜 |
| 客流来源 | 熟人 + 社区 + 学生 | 上班族 + 路人 |
| 营销 | 微信群 + 老带新 | 团购 + 抖音 |

> **重要原则**：**不要把一二线的品类直接复制到县城**——这是中年破产四件套的根源之一。

## G. 火锅 / 烤鱼 / 早餐 / 牛排 / 面馆专项

### G.1 火锅
- 100㎡ 以上才有底料 + 小料经济性
- 客单 60+ 才能覆盖人工
- **新店期 1 年内不要碰高端潮玩 / 主题火锅**

### G.2 烤鱼
- 等位时间长是生意好，但流量天花板低
- 适合做大店 + 商场店

### G.3 早餐
- 出餐速度 > 一切
- 一定要在通勤路径上
- 客单价 5–15 元为主，靠量

### G.4 牛排
- 西式平价牛排（豪客来式）已被快招玩烂
- 高端牛排店 → 重资产警告

### G.5 面馆
- 刚需 + 复购 + 标准化 → 好品类
- 但门槛低导致同行密集，必须有差异化（汤头 / 浇头 / 配套）

## H. 社交餐饮 vs 工具餐饮

```
社交餐饮：人来吃饭是为了"在一起"（聚餐、宴请、约会）
工具餐饮：人来吃饭是为了"填饱肚子"（解决三餐）
```

- 疫情后**社交餐饮萎缩**（高客单、低频次、强场景依赖）
- **工具餐饮稳定增长**（低客单、高频次、抗周期）
- **小白创业首选工具餐饮**

## I. 决策树：你应该开什么？

```
你有多少钱？
├─ < 5 万 → 摆摊 / 推车（烤面筋、煎饼、鸡叉骨）
├─ 5–15 万 → 小吃店（30㎡内，米粉/快餐/早餐）
├─ 15–30 万 → 中型快餐店 或 加盟蜜雪类大品牌
├─ 30–60 万 → 自营特色快餐 / 品牌面馆
└─ > 60 万 → 你不该自己摸索，先找专业顾问

你有几年餐饮经验？
├─ 0 年 → 优先快餐 / 早餐 / 面馆 / 摆摊
├─ 1–3 年 → 复制熟悉品类 + 微创新
└─ 3 年以上 → 可以挑战正餐或品牌化

你的目标客群？
├─ 学生 → 米粉 / 鸡排 / 奶茶（注意红海）
├─ 上班族 → 快餐 / 早餐 / 简餐
├─ 家庭 → 卤肉饭 / 砂锅 / 面馆
└─ 中老年 → 馄饨 / 早餐 / 平价小炒
```
