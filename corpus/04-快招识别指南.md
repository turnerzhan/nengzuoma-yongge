# 04 · 快招识别指南

> 快招（快速招商加盟）公司是勇哥直播间最高频的话题。**90% 的连麦悲剧背后都站着一家快招公司。**

---

## A. 什么是快招

**本质**：以短期牟利为核心的加盟诈骗模式。

**核心商业逻辑**：
> **用低门槛勾人，用假承诺骗签，用强绑定吸血，用快跑路收尾。**

它不是要你的店赚钱，**它是要你的加盟费、装修款、设备款、首批物料款**。一次性榨干你，就开始忙下一个。

---

## B. 快招公司的标准 SOP（5 步收割）

### Step 1 · 搜索截流
- 在百度、抖音、小红书购买"蜜雪冰城加盟""瑞幸加盟"等关键词广告
- 当你搜索正规品牌时被截流到快招公司

### Step 2 · 话术转化
- "你想加盟的品牌**在当地招满了 / 水太深了**，我推荐你一个更好的"
- "我们这是**蜜雪冰城的子品牌 / 平替 / 升级版**"

### Step 3 · 包装忽悠
- 装修精美的样板间总部
- 墙上挂满锦旗、证书、央视广告合影
- 雇人排队制造假繁荣
- 派人开"已开店主分享会"

### Step 4 · 邀约逼单
- 报销来回机票
- 保姆车接送，称呼你"X 总"
- 端茶递水
- 制造紧迫感："今天交定金有优惠 / 区域代理只剩最后一个"
- 一次性收：加盟费 + 设备费 + 物料费 + 装修费 + 培训费

### Step 5 · 售后消失
- 开店后发现产品毫无竞争力
- 联系总部：要么忽悠你"再加配菜 / 再加设备就好了"，要么人去楼空
- 你的店就是**他们财报里已结算的一笔订单**

## C. 快招识别信号清单

### C.1 信源类信号
- 总部不是在品牌官网找到的，而是通过**搜索引擎广告 / 短视频私信**看到的
- 本来想加盟 A 品牌，被告知"招满了"转而推荐你 B 品牌
- 添加你的销售微信里**只发自家品牌的"加盟成功案例"**

### C.2 地理信号（重灾区）
- 总部在 **济南**（最高警报）
- 总部在 **杭州 / 上海 / 郑州**（高警报）
- 总部所在地一个写字楼里**注册了 N 个不同品牌的公司**

### C.3 品牌信号
- 品牌名从没听过
- 全国门店数极少（或都集中在偏远地区）
- 拿不出**3 年以上稳定经营的直营店**

### C.4 话术信号（三句直接转身走）
- "**零加盟费**"
- "**6 个月回本**"
- "**总部全包 / 保姆式扶持**"

### C.5 商业条款信号
- 设备和物料**远高于市场价且必须从总部采购**
- 合同绑定条款极多（强制进货、超高违约金、违约不退保证金）
- 装修必须用**总部指定装修公司**

### C.6 工商信号（企查查 / 天眼查）
- 公司成立时间**< 2 年**
- **社保参保人数为 0** 或个位数
- 短时间内注册多个相似品牌
- 法人有大量历史诉讼或失信记录
- 经营异常 / 行政处罚记录

## D. 防坑核心金句

> **遇到"零加盟费""6 个月回本""总部全包"这三句话，直接转身走。真能赚的品牌，不会靠低门槛拉人，只会挑有能力的加盟商。**

> **当你连蜜雪的官方加盟电话都找不到，从那一刻起，结局就已注定。**

> **真品牌选你，假品牌求你。**（衍生）

---

## E. 查验品牌真伪的标准动作

### E.1 自己输入官网网址
- 不要点搜索引擎里**带"广告"标识**的链接
- 不要点抖音 / 小红书的私信
- 直接输入「品牌名 + 官网」核对域名年龄

### E.2 工商查询
- 企查查 / 天眼查查公司基本信息
- 重点看：成立年限、社保人数、经营状态、关联企业

### E.3 真实门店探访
- 不去总部样板店
- 自己挑一家**已开 1 年以上的加盟店**，到店里待半天
- 跟老板要一杯水，聊一聊：当初投了多少？现在一天卖多少？后悔吗？

### E.4 多渠道交叉验证
- 美团 / 大众点评看真实评价（不只看星级，看一星二星评价）
- 在抖音搜「品牌名 + 加盟 + 后悔 / 转让」
- 知乎搜该品牌讨论（特别看负面回答）

## F. 「哪吒仙饮」类型的新型快招特征（2025–2026 升级版）

老快招用样板间，新快招升级了：

1. **借势 IP**：蹭热门动画 / 影视 / 名人 IP（哪吒、孙悟空等）
2. **借势宝妈群**：在妈妈群里投放"宝妈在家也能创业"广告
3. **送车拓客**：你交 40 万加盟费 → 送你一辆 10 万车（车款已含在加盟费里）
4. **预包装备案当资质**：拿一张食品备案证当"国家认证"
5. **AI 生成内容**：用 AI 生成大量正面宣传视频，搜索结果被刷屏

## G. 快招重灾品类清单

| 品类 | 高发原因 |
|---|---|
| 杂牌奶茶 | 蹭蜜雪流量最容易 |
| 杂牌咖啡 | 蹭瑞幸 / 库迪流量 |
| 第 N 代汉堡 | 蹭塔斯汀 / 华莱士 |
| 卤味 / 鸭脖 | 易标准化 + 加盟需求大 |
| 中药养生饮品 | 借「养生」概念新酒装 |
| 玻尿酸 / 胶原蛋白饮品 | 借「美容」概念 |
| 概念性新品类 | 名字越离谱越要警惕 |

## I. 五步反诈检查表（skill 可直接调用）

用户告诉你他要加盟某品牌，你按这五步过一遍：

| # | 检查项 | 触发动作 |
|---|---|---|
| 1 | 你怎么知道这个品牌的？ | 搜索引擎广告 / 短视频 → +1 风险 |
| 2 | 总部在哪？ | 济南 → +2 风险；其他重灾区 → +1 |
| 3 | 加盟费 + 设备费 + 装修费一次交多少？ | > 30 万且承诺包扶持 → +2 风险 |
| 4 | 全国直营店有几家？开了几年？ | 直营 < 5 家或都开 < 1 年 → +2 风险 |
| 5 | 是否提到"零加盟费 / 6 个月回本 / 总部全包"？ | 命中任一 → +2 风险 |

**得分 ≥ 5**：高度疑似快招，**强烈建议放弃**
**得分 3–4**：有疑点，**必须实地验证 + 工商核查**
**得分 ≤ 2**：相对可控，但**仍需常规尽调**
