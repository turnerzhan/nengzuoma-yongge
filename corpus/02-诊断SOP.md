# 02 · 诊断 SOP

> 这是勇哥方法论的"主控台"。所有连麦、所有判断都跑在这条流水线上。

---

## A. 诊断已开门店：「这家店能不能救？」

### A.1 四步诊断流程

#### Step 1 · 经营状况梳理（定底线）

逐项询问，**每问一句记一个数**：

- 日营业额是多少？
- 毛利率多少？（很多店主答不上来，那就问"食材一天用多少钱"）
- 月房租多少？
- 人工成本多少？（几个员工，月薪各多少）
- 水电杂费多少？

→ 算出 **月度固定支出总额**
→ 算出 **保本线 = 月度固定支出 ÷ 毛利率**
→ 换算 **日保本营业额 = 月保本线 ÷ 30**
→ 对比实际日营业额，判断是否在「斩杀线」以下

#### Step 2 · 投资与背景调查（定抗风险能力）

- 总投资多少？怎么分布的？（加盟费 / 装修 / 设备 / 物料）
- 钱是自有资金还是贷款 / 借债？有没有抵押房产？
- 加盟还是自营？如果加盟，**品牌方是谁？总部在哪？**
- 有没有餐饮从业经验？之前做什么的？
- 家庭负担如何？有没有退路？

> 关键判断：**抵押房产 / 贷款创业 / 借亲友钱**——三大死亡信号，命中任一项立即提高风险等级。

#### Step 3 · 商圈与环境评估（定客流上限）

经典操作：「**来，镜头高一点，360 度原地转个圈**」

通过实景视频直观提取：
- **客流质量**：住宅区 / 商业街 / 学校 / 写字楼 / 荒郊野岭？
- **周边业态**：旁边都开了什么店？有没有大品牌（蜜雪、瑞幸、塔斯汀、华莱士）？
- **竞争环境**：同品类有几家？他们生意怎么样？
- **周边数据**：「周围的店一天卖多少钱？」
- **人流量实测**：有没有在门口坐一天数过人头？

> 详细 12 个观察点见 [`skill/街景观察清单.md`](../skill/街景观察清单.md)

#### Step 4 · 三维交叉验证 → 结论

三个维度必须**全部匹配**才能"可做"：

| 维度 | 核心问题 | 工具 |
|---|---|---|
| 成本核算 | 日保本线是多少？ | 保本线公式 |
| 投资背景 | 抗风险能力多强？ | 资金来源 + 家庭情况 |
| 商圈环境 | 客流上限是多少？ | 360° + 蹲点数人流 |

### A.2 三档结论

#### 档位 1：直接劝退型 → "关了吧，今天就关。"

触发条件（任一）：
- 选址根本性错误（如二楼杂牌奶茶 / 偏远工业区 / 综合体三楼）
- 产品与商圈严重不匹配（如蜜雪旁边开杂牌果汁）
- 深陷加盟骗局（快招特征明显）
- 高负债 + 日营业额低于保本线 50%

经典判断：长治「七层奶茶大厦」 → 日销 800 覆盖不了成本，高负债扛不住，必倒闭。

#### 档位 2：整改尝试型 → 给优化建议 + 设观察期

触发条件：位置尚可但经营有问题
- 给具体优化建议（菜单 / 价格 / 营销）
- 设硬性观察期（如「1 个月内流水未达 X 再闭店」）

经典判断：山东果汁店 → 保本 1600 > 日销 500，先做低价引流，1 个月未达 1200 再闭店。

#### 档位 3：可行优化型 → 差异化策略

触发条件：保本达标，有提升空间
- 给差异化策略建议
- 给增长引擎建议（外卖 / 团购 / 复购钩子）

经典判断：江苏李姐面馆 → 保本达标，通过「免费续面 + 加微信送卤蛋」做差异化。

## B. 判断新店能不能开：「开店五步法」

### Step 1 · 选品

- 卖什么？为什么选这个品类？
- 是**刚需品类**还是非刚需？（猪脚饭、葱油饼 > 奶茶、咖啡、烘焙）
- 目标客群是谁？消费能力匹配吗？

> **中年破产四件套：奶茶、汉堡、咖啡、烘焙** —— 红海，小白慎入

### Step 2 · 投资预算

- 总预算多少？来源？
- **绝对禁忌**：贷款创业 / 抵押房产创业 / 借亲友钱创业
- 投资要控制：**轻资产启动**，装修不要过度
- 钱应该花在哪：核心设备 > 食材储备 > 装修门面

### Step 3 · 商圈选址（最核心）

「选址看人流量，**别凭感觉**」

**必做功课**：
1. 在目标铺位前坐**至少一整天**，用手机统计人流量
2. 看周边店铺的营业情况作为参考基线
3. 警惕：商住综合体、新开发区、鬼城楼盘
4. 看 [`skill/街景观察清单.md`](../skill/街景观察清单.md) 里的 12 个观察点

> **经典反问**：浑水做空瑞幸都是派人去门店数小票的，**你连花一天数人头都做不到，就敢投半辈子积蓄？**

### Step 4 · 成本盈利预测

- 预估月固定支出（房租 + 人工 + 水电 + 杂费）
- 预估毛利率（参见各品类参考）
- 算出保本线营业额
- 对比商圈内同业态的平均营业水平
- 如果**保本线 > 商圈平均水位** → 高风险

### Step 5 · 风险预案

- 如果 3 个月达不到保本线怎么办？
- 最大能承受亏损多少？亏到什么程度必须止损？
- 设备和装修能回收多少？（通常大幅贬值）
- 有没有 plan B？

## C. 保本线核心公式

```
利润 = 营业额 - 食材成本 - 固定费用

毛利 = 营业额 - 食材成本
毛利率 = 毛利 / 营业额 × 100%

固定费用 = 房租 + 人工 + 水电 + 杂费

保本线（月） = 固定费用 / 毛利率
保本线（日） = 保本线（月） / 30

斩杀线 = 实际日营业额 / 日保本线 < 60%
```

> 详细计算器与各品类参数见 [`skill/保本线计算器.md`](../skill/保本线计算器.md)

## D. 勇哥独家：小票计算法（不用问就能算别人营业额）

> 教你估别人店里一天卖多少：

1. 早上开门蹲守，记下**第一张小票号**
2. 关门前再去，记下**最后一张小票号**
3. 两者相减 = 当天单数
4. 单数 × 估算客单价 = 当日营业额

适用：开放式厨房 / 收银小票连号的快餐店、奶茶店。

## E. 八连问（标准化提问公式）

勇哥连麦冷启动用的标准提问，可作为 skill 提问树骨架：

1. 你好哥们，遇到什么问题了？
2. 开店前认识我吗？
3. 花了多少钱？
4. 公司总部在哪？
5. 去过总部吗？
6. 有店还是只有样板间？
7. 关了吧，今天就关？
8. 该起诉起诉。

> 完整提问树状态机见 [`skill/提问树.md`](../skill/提问树.md)

---

## F. 四连问（精简版诊断）

适合时间有限的快速判断：

1. 卖的是啥？
2. 一天卖多少钱？
3. 毛利率多少？
4. 租金多少？

→ 4 个数字一齐，**保本线立刻浮现**。
