# 03 · 选址方法论

> 选址在勇哥的体系里是 **决定餐饮成败 80% 的变量**。这一章是全仓库密度最高的一章。

---

## A. 四大选址模型

```
匹配商圈 ─→ 聚客街道 ─→ 优质铺位 ─→ 投入产出
（宏观） （中观） （微观） （财务）
```

### A.1 匹配商圈
- 你的产品价格带匹配这个商圈的主流消费水平吗？
- 县城 8 元粥铺开在 CBD 写字楼下？错配
- 28 元客单价奶茶开在城中村？错配
- **客单价 ÷ 商圈人均消费 ≈ 0.6–1.2** 为合理区间

### A.2 聚客街道
- 这条街是「**有自己主题**」还是「随便几家店凑成的街」？
- 美食街 / 小吃街 / 学校外街 / 早夜市 = 聚客街
- 商住底商沿街 / 新开发区主干道 = 通常不聚客
- **核心判断**：周末 / 晚饭点，行人是「奔着这条街来」还是「路过」？

### A.3 优质铺位
- "金角银边草肚皮" —— **角铺 > 边铺 > 中段铺**
- 门头宽度 ≥ 1.2 米
- 一楼临街，门口能看见
- 易接近度高（无台阶 / 无栏杆 / 无绿化带遮挡）

### A.4 投入产出
- 转让费 + 装修 + 设备 + 物料 + 半年现金流 ≤ 你能承受的总数？
- 月房租 ≤ 月预估营业额 × 15%？
- 12–18 个月内能回本吗？

---

## B. 转让费三大评估标准（硬核）

接手别人的转让铺，转让费给多少合适？勇哥给了三条铁律：

### B.1 标准一：3–4 个月赚回原则
- 你预估月净利润 × 3–4 个月 ≥ 转让费 → 可接受
- 如果要 1 年以上才能赚回 → 转让费虚高，**不划算**

### B.2 标准二：≤ 月租 × 15 倍
- 转让费 ≤ 月房租 × 15 → 合理
- 比如月租 8000，转让费 12 万以上 → 偏贵
- 这条用来快速过滤"狮子大开口"

### B.3 标准三：当月营业额 ≥ 转让费 ÷ 2
- 你接手第一个月做出来的营业额 ≥ 转让费一半 → 转让费合理
- 否则现金流压不住

### B.4 三类不能碰的转让铺
1. **私下转让**（绕开房东、没合同保障）
2. **二房东收转让费 + 租金差价**（双重剥削）
3. **带「品牌技术」溢价的转让**（多半是甩包袱 + 想拉你下水）

## C. 人流量测算与小票法

### C.1 蹲守数人流
- **工作日 2 天 + 周末 1 天**（共 3 天）
- 每天分 3 个时段：早餐 / 午餐 / 晚餐
- 用手机计数器，记录目标客群（不是所有路人）

### C.2 进店率经验值
- **正常进店率 ≈ 3%**（路过 100 人有 3 人进店）
- 用进店率反推日客单数 = 每日有效路过 × 3%
- 日营业额 ≈ 日客单数 × 客单价

### C.3 小票计算法（估别人店）
- 早上记第一张小票号，晚上记最后一张
- 单号差 = 单数；× 客单价 = 营业额
- 适用：开放厨房、连号小票的快餐店奶茶店

### C.4 房租营业额铁律

> **1 万房租对应日营业额应该在 3000 元左右**

倒推公式：
```
合理日营业额 ≈ 月房租 × 0.3
合理月营业额 ≈ 月房租 × 9（即房租占比约 11%）
```

如果你的预估营业额低于这个比值 → 房租太贵或选址错。

## D. 同业浓度黄金区间

> 「**40–120 家**」是同品类小吃在一个商圈的健康浓度。

- < 40 家：商圈不成熟，客流没养起来，先驱者多半成先烈
- 40–120 家：商圈已成熟，流量充足，**新店有空隙生存**
- > 120 家：内卷，进去就要打价格战 / 拼装修 / 拼营销

判断方法：用美团 / 大众点评搜目标商圈 + 品类，数门店数。

## G. 阴阳街判定法

- **阳街**：朝南 / 朝东，白天有阳光，门面亮堂，人愿意停留
- **阴街**：朝北 / 朝西被楼挡，全天阴冷，过路客流不停留
- 同一条街两侧租金可能差 30–50%，但很多人只看租金便宜不看朝向

## H. 销售推动业态叠加法

判断一个商圈的客流上限：

```
有效客流 ≈ Σ（周边业态客流密度 × 距离衰减）

写字楼（200米内）：×1.0
学校（300米内）：×0.8
大型小区 1500 户以上（500米内）：×0.6
商场主力店（同栋楼）：×0.7
```

**叠加 ≥ 2 个销售推动业态** → 商圈活；只有 1 个 → 单点依赖风险大；都没有 → 别开。

## I. 双证 + 三排查（开店前必须确认）

- **双证**：营业执照 + 食品经营许可证（**没有，不要租**）
- **三排查**：
- 油烟（有没有正规烟道？小区楼下尤其要查）
- 排水（有没有隔油池？后厨排水接得通吗？）
- 用电（有没有商业电？三相电？大功率设备能装吗？）

> 这三项任何一个出问题，可以让一家店开不起来或开起来天天被投诉。

---

## J. 「找店像打井」哲学

> **找店像打井：一米不出水就换地，永远打不出好井。**

意思是：**选址要有耐心**，看到看上的位置必须严谨判断（最好蹲点 + 测算 + 比较至少 3 个备选铺位）。但一旦认定就坚定执行；一直浅尝辄止换地点，最终只会把启动资金消耗在中介费和加盟费里。
