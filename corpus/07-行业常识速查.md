## B. 各项成本占比健康区间

| 成本项 | 占营业额比 | 危险阈值 |
|---|---|---|
| 食材成本 | 25%–45%（视品类） | > 50% |
| 房租 | 8%–15% | > 20% |
| 人工 | 15%–25% | > 30% |
| 水电杂费 | 3%–5% | > 8% |
| 营销 | 2%–5% | 长期 > 8% 不健康 |
| **净利率** | **10%–20%** | **< 5%** |

> **三大警戒线**：房租 > 20% / 食材 > 50% / 净利率 < 5%，命中任一项 → 必须立刻整改。

## E. 翻台率与坪效

```
翻台率 = 每日就餐桌次 / 总桌数
坪效 = 每月营收 / 营业面积（㎡）
人效 = 每月营收 / 员工数
```

| 模式 | 翻台率 | 坪效 |
|---|---|---|
| 快餐 | 1.5–3 次/日 | 1500–3000 元/㎡/月 |
| 正餐 | 0.8–1.5 次/日 | 1000–2000 元/㎡/月 |
| 火锅 | 1–2 次/日 | 1200–2500 元/㎡/月 |
| 饮品 | — | 看出杯数 |

---

## F. 客单价与日营业额估算

```
日营业额 = 日客单数 × 客单价
日客单数 = 日有效路过 × 进店率（约 3%）
月营业额 = 日营业额 × 30
```

**经验值**：
- 路过 → 进店率 ≈ 3%
- 进店 → 下单率 ≈ 60%–80%
- 老客回头率 ≈ 30%–50%（健康刚需餐饮）

---

## G. 房租与营业额匹配铁律

> **1 万房租对应日营业额应在 3000 元左右**

衍生公式：
```
合理日营业额 ≈ 月房租 × 0.3
合理月营业额 ≈ 月房租 × 9（房租占比约 11%）
```

**对照表**：
| 月房租 | 合理日营业额 | 合理月营业额 |
|---|---|---|
| 5,000 | 1,500 | 4.5 万 |
| 8,000 | 2,400 | 7.2 万 |
| 1 万 | 3,000 | 9 万 |
| 2 万 | 6,000 | 18 万 |
| 3 万 | 9,000 | 27 万 |
| 5 万 | 15,000 | 45 万 |

## J. 餐饮闭店潮数据（背景）

- 2024 年中国餐饮门店净关店约 **354 万家**（公开报道）
- 平均每家餐饮店生命周期约 **18 个月**
- 加盟店第一年关店率约 **30%–40%**（依品类）
- 三年存活率约 **15%–25%**
