## E. 案例使用建议（给 skill）

skill 在生成回复时，应：
1. **先收集用户的 4–8 个核心数字**（参见 [`../corpus/02-诊断SOP.md`](../corpus/02-诊断SOP.md)）
2. **按金额段位 + 品类 + 选址类型** 匹配最相近案例
3. **类比方式输出**：「跟你最像的是 XXX，那哥们当时也是…… 最后…… 」
4. **附带勇哥原话锚点**（参见 [`../corpus/06-金句与话术.md`](../corpus/06-金句与话术.md)）

> **类比比说教更有杀伤力**——这是勇哥连麦最锋利的武器。
