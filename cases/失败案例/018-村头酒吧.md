# 案例 018：村头酒吧

> 一句话摘要：16 万开酒吧，爹妈当服务员，10 天亏光。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 018 |
| 当事人 | 一位年轻创业者 |
| 时间 | 2025 年 |
| 地点 | 农村 |
| 品类 | 酒吧 |
| 投资金额 | 16 万 |
| 经营时长 | 10 天 |
| 最终结局 | 亏光全部投入 |

## 起因

借鉴县城酒馆短视频热度，把"酒吧概念"搬到了村头。家里人担心他亏，主动来当服务员省人工。

## 经过

- 选址村头，无主流酒吧客群
- 概念太"潮"，村民不习惯也不消费
- 父母在店里值班，但客流过低
- 10 天亏光 16 万

## 勇哥诊断

- 商圈错配是头号问题：村里没有酒吧客群
- 把"潮玩"搬到不潮的地方 = 自说自话
- 让父母当服务员 = 心理上的最后求救
- 关，越早越好

## 失败原因

1. 商圈与品类严重错配
2. 客群购买力与消费习惯不支持
3. 家庭成员充当人力 = 现金流危机指标
4. 短视频幻觉："县城酒馆火了"≠"村头也能火"

## 给 skill 的启示

- 用户提到「农村 / 县城以下 / 镇上」+ 非刚需主题餐饮 → 直接劝退
- 家庭成员临时充当员工是危险信号，应单独提示
- "短视频火过的品类不代表你这个地点也能火"
