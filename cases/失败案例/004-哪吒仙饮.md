# 案例 004：哪吒仙饮 · 宝妈陷阱

> 一句话摘要：借哪吒 IP 包装的概念奶茶，2025 年底引爆勇哥直播间。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 004 |
| 当事人 | 多名加盟商（含大量 00 后宝妈） |
| 时间 | 2025 年下半年 |
| 地点 | 全国 |
| 品类 | 概念奶茶 |
| 投资金额 | 单店多在 20–40 万区间 |
| 资金来源 | 普遍含贷款 / 借款 |
| 经营时长 | 多数尚未回本即陷困 |
| 最终结局 | 多数处于亏损或观望 |

## 起因

公司 2022 年成立，2025 年 7 月才完成预包装食品备案，社保参保人数为零。借「哪吒」热点话题包装，通过短视频平台投放加盟广告，目标人群明确指向 00 后宝妈群体。

## 经过

- 用「全民国潮、亲子友好」等话术包装
- 短视频里反复出现"宝妈在家也能创业"的引导
- 部分加盟商已开店数月，普遍营业额低迷
- 有加盟商谎报营业额试图找下家接盘，被勇哥连麦时当场识破
- 网友反讽："我不喝蜜雪、瑞幸、霸王茶姬，就要喝哪吒仙饮"

## 关键数据

- 公司成立年限：约 3 年
- 食品备案时间：2025 年 7 月
- 社保参保人数：0
- 加盟商背负负债：常见数十万级

## 勇哥诊断

- 该起诉起诉
- 真品牌选你，假品牌求你
- 当一家公司连参保人都没有，你还指望他对加盟商负责？

## 失败原因

1. 蹭 IP 借势，缺乏长期品牌沉淀
2. 公司主体过于"轻"，社保为零是工商层面的红色信号
3. 主攻宝妈群体，是风险识别能力较弱的画像
4. 概念噱头无复购支撑

## 给 skill 的启示

- 用户提到任何「IP 名 + 仙饮 / 茶饮 / 奶咖」组合时，触发 D 线五步反诈检查
- 对宝妈群体语气需更克制，但仍要直说真话
- 引用工商查询要点：成立年限、社保人数、关联企业

## 出处

- 抖音「勇哥餐饮创业说」连麦切片
- 网易、新浪、腾讯新闻多家跟进报道
- 企查查公开数据
