# 案例 003：第四代汉堡 · 宁夏 6 天 90 万

> 一句话摘要：扬言"拳打麦当劳脚踢华莱士"，开业第 6 天花篮还在门口就贴了转让。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 003 |
| 当事人 | 宁夏贺女士 |
| 时间 | 2025 年 |
| 地点 | 宁夏 |
| 品类 | 杂牌"第四代汉堡"（快招品牌） |
| 投资金额 | 90 万 |
| 资金来源 | 含父亲赔偿款 |
| 经营时长 | **6 天** |
| 最终结局 | 全部赔光，转让 |

## 起因

宁夏贺女士被快招公司话术打动，加盟一个号称"第四代汉堡"的杂牌品牌。

广告语："**拳打麦当劳肯德基、脚踢塔斯汀华莱士**"

并被忽悠拿下当地代理权——典型**庞氏代理结构**：让你交一大笔代理费，再让你去发展下线加盟商。

## 经过

- 加盟费：32 万
- 物料费：8 万
- 设备费：18 万
- 仅交给总部部分：**58 万**
- 加上装修 / 房租 / 流动资金，总投入 ~90 万
- 开业第 6 天发现品牌毫无吸引力，营业额惨淡
- 第 6 天：**花篮还在门口就贴了"转让"**

## 关键数据

- 总投资：90 万
- 给总部部分：58 万（加盟 32 + 物料 8 + 设备 18）
- 经营天数：6 天
- 6 天营业额：极少（具体数字未公开）
- 单日亏损：约 15 万 / 天

## 勇哥诊断

> 「**该起诉起诉。**」

> 「你想加盟麦当劳，对方告诉你招满了，推荐你一个'第四代汉堡' —— 哥们儿你这一步走的，结局就已经写好了。」

> 「**当你连蜜雪 / 麦当劳的官方加盟电话都找不到，从那一刻起，结局就已注定。**」

## 失败原因分析

1. **典型快招广告**：搜索截流 + 话术包装
2. **庞氏代理陷阱**：被忽悠拿代理权，注定无人接盘
3. **品牌无价值**："第四代汉堡"全无认知度
4. **设备物料溢价**：18 万设备实际市场价可能不到一半
5. **决策周期过短**：从了解到打款不到 1 个月
6. **资金结构悲剧**：含**父亲赔偿款**，背后是更深的家庭悲剧

## 给 skill 的启示

- **当用户描述：加盟一个"第 N 代 / 升级版 / 子品牌"汉堡 / 奶茶 / 咖啡** → 几乎可以立即判定快招
- **当用户提到"区域代理 / 总代理 / XX 省独家"** → 庞氏结构警报
- **当用户提到资金来源是父母赔偿款 / 拆迁款 / 急用钱** → 情感+理性双重劝退
- 输出建议：**起诉，全力追讨。同时不要再追加任何投资。**
