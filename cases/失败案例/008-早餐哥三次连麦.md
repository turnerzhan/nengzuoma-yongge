# 案例 008：早餐哥 · 三次连麦

> 一句话摘要：包子→火锅→面馆，三次创业三次找勇哥连麦。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 008 |
| 当事人 | 早餐哥 |
| 时间 | 2023 / 2024 / 2025 |
| 地点 | — |
| 品类 | 包子铺 → 火锅店 → 面馆 |
| 经营时长 | 每次都没撑过半年 |
| 最终结局 | 三连失败 |

## 起因

最早一次开包子铺，被勇哥提醒选品和位置都不行；
未听劝，又开了火锅店；
火锅亏掉之后又开面馆，再次连麦。

每次都说"这次不一样"，每次都重复同样的错误。

## 经过

第一次（包子铺）
- 选址不佳 + 投入控制不力
- 勇哥已点出问题
- 当事人未听劝

第二次（火锅店）
- 跨品类大幅提高复杂度
- 高投入 + 高人工 + 高房租
- 再次失败

第三次（面馆）
- 表面回归刚需品类
- 但仍存在选品 / 选址 / 节奏问题
- 第三次连麦被勇哥提及"懒散是核心问题"

## 勇哥诊断

- 干要苦干，不要幻想
- 你不是输在品类，是输在你自己
- 餐饮老板要亲自干，不能当甩手掌柜
- 每天上一当，当当都一样

## 失败原因

1. 老板姿态：长期想"动手少、收益多"
2. 不听劝 + 反复犯错
3. 每次都跨品类，无法积累认知和供应链
4. 选址在每一次都不达标

## 给 skill 的启示

- 用户描述"曾失败过 N 次，这次想重新开始"时 → 重点提示是否反思过老板自身问题
- 跨品类创业 = 高风险，必须给出明确警告
- 每个新品类回到 C 线五步法重新评估，不要假设"以前的经验有用"

## 出处

- 抖音「勇哥餐饮创业说」连麦切片
- 多家媒体报道
