# 案例 002：七层奶茶大厦 · 长治"水云间"

> 一句话摘要：90 万抵押贷款，在县城边缘租整栋 5 层清水楼开杂牌奶茶。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 002 |
| 当事人 | 长治大哥 |
| 时间 | 2025 年 |
| 地点 | 山西长治（县城边缘） |
| 品类 | 杂牌奶茶 + 棋牌 + 零食（混合业态） |
| 投资金额 | 90 万+（30 万积蓄 + 60 万贷款） |
| 资金来源 | 自有 30 万 + **抵押房产贷款 60 万** |
| 经营时长 | 几周内日销 800 |
| 最终结局 | 重资产高负债下基本判定必倒 |

## 起因

长治大哥本来想加盟蜜雪冰城。被搜索引擎广告截胡后，被快招公司忽悠"蜜雪招满了，我们这是蜜雪子品牌"，最后选了一个**杂牌奶茶**。

为了"做大做强"，租下一栋 5 层清水楼整体改造。

## 经过

- 选址：**离县中心开车 10 分钟的边缘地段**
- 业态规划：
- 1–2 层：奶茶
- 3–4 层：棋牌室
- 5 层：零食店
- 楼顶：网红打卡点
- 装修：清水楼全部重做，巨大投入
- 资金：自有 30 万 + 房产抵押贷款 60 万
- 开业后：日销售 800 元
- 旁边：有正规蜜雪冰城

## 关键数据

- 总投资：90 万+
- 月房租：估计 ≥ 3 万（5 层清水楼）
- 月人工：估计 ≥ 1.5 万（多业态多员工）
- 月固定支出：保守估计 ≥ 5 万
- 假设毛利率 70% → **月保本线 ≈ 7.1 万 → 日保本 ≈ 2400 元**
- 实际日销：800 元
- **亏损率**：保本达成 33% → **远低于斩杀线（60%）**
- 月亏损：估计 4–5 万

## 勇哥诊断

> 「**这钱够你开 3 家正规蜜雪冰城。**」

> 「你这店从你开的那一刻就已经死了，只是过了几个月才埋。」

> 「位置在生意在，位置被搞走生意就没。你这位置 —— 没。」

> 「县城里 5 层楼网红打卡？你不是在开店，你是在搞房地产开发。」

## 失败原因分析

1. **典型快招截胡**：从蜜雪被骗到杂牌
2. **重资产病**：5 层清水楼装修，把启动资金全埋进去
3. **抵押房产创业**：触发勇哥头号警告（绝对禁忌）
4. **错估业态协同**：以为多业态能引流，实际是多业态多分摊房租
5. **错配选址**：县城边缘 + 多业态网红 = 没有自然客流
6. **资金结构灾难**：60 万贷款带来的月供利息直接压垮现金流

## 给 skill 的启示

- **当用户描述：投资 > 50 万 + 抵押房产 + 县城/边缘 + 多层楼** → 立即最高级别警报
- **当用户提到"网红打卡"+"多业态"+"小县城"** → 红色警报
- **当用户原本想加盟大牌但被推荐"子品牌"** → 触发快招识别（参见 [`../../corpus/04-快招识别指南.md`](../../corpus/04-快招识别指南.md)）
- 输出建议：**关，立刻关。趁还没把所有装修和设备贬值掉。**

## 出处

- 抖音「勇哥餐饮创业说」连麦切片
- 多家媒体跟进报道（虎嗅 / 网易 / 36 氪）
- 网友戏称"水云间"
