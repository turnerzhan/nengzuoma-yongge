# 案例 021：韩国咖啡 · 160 万

> 一句话摘要：一辆迈巴赫的钱亏在了一家小咖啡店里。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 021 |
| 当事人 | 一位创业者 |
| 时间 | 2024–2025 |
| 品类 | 韩系咖啡 |
| 投资金额 | 约 160 万 |
| 最终结局 | 重大亏损 |

## 起因

被"韩国咖啡 = 高级、文艺、高客单"概念吸引，进口设备 + 重装修 + 高端选品全套上车。

## 经过

- 进口咖啡机数十万
- 重装修 + 进口家具
- 客单价远高于瑞幸 / 库迪
- 与 9.9 元价格战正面碰撞
- 慢慢亏到 160 万

## 勇哥诊断

- 你这家店从开起来那一刻就在送钱
- 咖啡赛道核心是性价比，不是文艺
- 瑞幸库迪 9.9 把利润空间压死了
- 一辆迈巴赫的钱够你试错好几年的事，全埋了

## 失败原因

1. 重资产病：进口设备 + 进口家具 + 重装修
2. 价格定位错配：在 9.9 时代做高端
3. 复购薄弱：文艺打卡型客流
4. 沉没成本拒绝面对：亏掉一半才开始连麦

## 给 skill 的启示

- 用户提到「韩系 / 日系 / 欧式咖啡 / 进口设备」→ 默认重资产警报
- 与 9.9 价格战正面战场的咖啡 → 高风险
- "文艺感"不是一个商业模型
