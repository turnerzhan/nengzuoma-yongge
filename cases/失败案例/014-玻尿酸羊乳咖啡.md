# 案例 014：玻尿酸羊乳咖啡 · 上海

> 一句话摘要：交 40 万加盟费送一辆车，新型快招的进化版。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 014 |
| 当事人 | 一位上海创业者 |
| 时间 | 2025 年 |
| 地点 | 上海 |
| 品类 | 概念咖啡（玻尿酸 + 羊乳） |
| 投资金额 | 40 万（加盟费已含赠车价值） |
| 最终结局 | 概念无人接，复购薄弱 |

## 起因

加盟方主打"美容 + 健康"双概念，在销售环节用"加盟即送车"刺激下单决策。

实际车价已经被打入加盟费，相当于自掏腰包给自己买了辆 10 万的车。

## 经过

- 加盟费 40 万，附赠"价值 10 万"的车
- 概念噱头："喝咖啡补玻尿酸"
- 客户接受度低
- 复购几乎为零
- 加盟方在售后阶段持续推销新物料

## 勇哥诊断

- 送车型快招是新版本套路
- 你以为占便宜，其实车价就在加盟费里
- 玻尿酸 + 羊乳 + 咖啡，每一个组合都很离谱
- 概念噱头 + 高客单价 + 低复购 = 必死

## 失败原因

1. 新型快招包装：用"礼物"掩盖加盟费虚高
2. 概念噱头无消费基础
3. 客单价高 + 复购低 = 现金流失血
4. 与瑞幸 / 库迪 9.9 元价格战正面对撞

## 给 skill 的启示

- 用户提到「加盟送车 / 加盟送装修 / 加盟送设备」→ 警惕"礼物"已被打入加盟费
- 任何"X 酸 / X 蛋白 / 健康概念 + 饮品"组合 → 默认快招怀疑
- 与瑞幸 / 库迪同战场的高客单咖啡品牌 → 默认劝退

## 出处

- 网易、搜狐 2025 年元旦盘点
- 抖音「勇哥餐饮创业说」连麦切片
