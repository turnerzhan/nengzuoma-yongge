# 案例 007：自嗨哥 · 北京减脂餐到小火锅

> 一句话摘要：逻辑无懈可击，不妨碍一个月就倒闭。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 007 |
| 当事人 | 北京大哥 |
| 时间 | 2024–2025 |
| 地点 | 北京 |
| 品类 | 高端减脂餐外卖 → 小火锅 |
| 投资金额 | 累计 100 万+ |
| 经营时长 | 减脂餐约 1 个月，火锅持续亏损 |
| 最终结局 | 连续两轮失败，仍在自我循环 |

## 起因

北京大哥认定"减脂餐是健康消费蓝海"，先做高端减脂餐外卖。

- 进店要先交 1000 元会员费
- 月租 1.5 万
- 大哥本人善于阐述商业逻辑，整体架构看起来"无懈可击"

## 经过

第一轮：减脂餐外卖
- 入会费门槛劝退绝大多数路人
- 月租 1.5 万但客单转化极少
- 一个月后倒闭

第二轮：小火锅
- 100 万投入，依然不听劝
- 持续亏损中

## 勇哥诊断

- 你说得很有逻辑、无懈可击，但不妨碍一个月就倒闭
- 你的问题不是商业模式，是你这人不听劝
- 第一次倒闭还可以说不熟，第二次还是这套就是认知问题

## 失败原因

1. **设置消费门槛 + 高价位**：入会费 1000 元等于直接劝退路人
2. **目标客群与商圈不匹配**：减脂高端客群本就少
3. **不听劝 + 自我循环**：连续两次创业都不愿听外部反馈
4. **沉没成本幻觉**：失败后想用更大投入找补

## 给 skill 的启示

- 用户表现出"我的逻辑很完美 / 我已经想清楚了"姿态时 → 警惕认知锁死
- 用户有"前一次失败"履历但仍要做类似品类时 → 红色警报
- 推荐路径：让用户先去同业态店蹲点 + 问真实老板
