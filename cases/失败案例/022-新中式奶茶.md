# 案例 022：新中式奶茶

> 一句话摘要：42 万开在一点点和蜜雪冰城中间，一边都打不过。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 022 |
| 当事人 | 一位创业者 |
| 时间 | 2025 年 |
| 品类 | 新中式概念奶茶 |
| 投资金额 | 42 万 |
| 最终结局 | 营业额低迷 |

## 起因

希望走"新中式 + 国风"差异化，在选址时认为"夹在两大牌中间能蹭客流"。

## 经过

- 选址：左侧蜜雪冰城，右侧一点点
- 客单价介于二者之间
- 蜜雪客群嫌贵不来
- 一点点客群觉得品牌不熟也不来
- 国风概念无法驱动转化

## 勇哥诊断

- 你这是给两边大牌做客流分母
- 客单介于中间 = 两边都不喜欢
- 国风概念在大众饮品赛道驱动力很弱
- 选址叫"夹缝", 不是叫"借势"

## 失败原因

1. 选址错配：自杀式夹缝定位
2. 客单错配：中间价位被两端挤压
3. 概念错配：国风对刚需饮品复购无加成
4. 投资过重：42 万对小奶茶店是上限

## 给 skill 的启示

- 用户提到「夹在两大品牌中间 / 隔壁都是 X」→ 必须问清楚谁是同品类
- 客单价应远离巨头主流价位带，避免正面碰撞
- 概念差异化不能替代真实需求差异化
