# 案例 020：北海万达集装箱

> 一句话摘要：押三付三的集装箱夜市铺位，季节性陷阱完整版。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 020 |
| 当事人 | 一位创业者 |
| 时间 | 2024 年 |
| 地点 | 北海万达集装箱夜市 |
| 品类 | 夜市小吃 |
| 最终结局 | 押金 + 装修难收回 |

## 起因

被夜市运营方推介"网红打卡 + 旅游城市夜经济"概念。

## 经过

- 押三付三：一次性占用大量现金
- 夜市运营方掌握主动权，规则随时调整
- 北海属于旅游城市，淡旺季差异巨大
- 装修无法带走，押金回收困难
- 实际经营时间被压缩

## 勇哥诊断

- 押三付三本身就是过滤新手的陷阱
- 旅游城市集装箱夜市 = 季节性 + 规则性双重不确定
- 装修扔在那是沉没成本
- 押金能不能回来全看运营方心情

## 失败原因

1. 资金结构灾难：押三付三吃掉流动资金
2. 季节性陷阱：淡季无法覆盖固定成本
3. 装修不可移动：沉没成本
4. 运营方主动权：规则调整不可控

## 给 skill 的启示

- 用户提到「夜市 / 集装箱 / 流动摊位运营方」→ 必须问押金条款
- 押三付三 = 黄牌警告
- 旅游城市旺季营收 ÷ 12 才是合理月营收估计
