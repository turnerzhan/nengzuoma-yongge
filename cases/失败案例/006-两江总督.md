# 案例 006：两江总督 · 江西大姐

> 一句话摘要：抵押住房 55 万拿下两省总代理，典型庞氏代理结构。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 006 |
| 当事人 | 江西大姐 |
| 时间 | 2025 年 |
| 地点 | 江西、湖南两省 |
| 品类 | 加盟饮品 |
| 投资金额 | 55 万 |
| 资金来源 | **抵押自住房** |
| 经营时长 | — |
| 最终结局 | 重大亏损，需自行招募下线消化压力 |

## 起因

某快招公司向其推介"两省总代理权"概念，大姐被"以代代加"模式打动，认为只要发展下线加盟商就能回本。

## 经过

- 抵押自住房贷款 55 万
- 一次性交清两省总代理费
- 之后被要求自行招商以回本
- 招商困难，库存积压
- 公司开始施压继续进货

## 勇哥诊断

- 抵押房产创业是头号死亡信号
- 区域代理 / 总代理本质都是把你变成下一个销售
- 真品牌不需要你帮他发展加盟商
- 现在唯一能做的就是止损 + 留证据

## 失败原因

1. 抵押住房：触发最高级警告
2. 庞氏代理结构：你就是被套现的中间环节
3. 缺乏招商能力却被要求承担招商任务
4. 信息不对称：不了解总部真实直营状况

## 给 skill 的启示

- 用户提到「区域代理 / 总代理 / 省代 / 市代」→ 立即触发庞氏结构警报
- 配合 D 线快招检查表
- 抵押住房 = 一票否决，无论项目多么"看上去合理"
