# 案例 009：星空烤鱼哥

> 一句话摘要：不听劝的典型，前后亏掉一百多万。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 009 |
| 当事人 | 星空烤鱼哥 |
| 时间 | 2024–2025 |
| 品类 | 主题烤鱼（"星空"主题装修） |
| 投资金额 | 100 万+ |
| 最终结局 | 重资产沉没 |

## 起因

希望走"主题餐饮"差异化路线，搞了一套星空主题灯光装修，单店投入接近百万。

## 经过

- 重装修：星空灯光 / 投影 / 沉浸式体验
- 烤鱼本身是高翻台率赛道，被主题设定降速
- 主题餐厅噱头大于实质，复购无支撑
- 多次连麦勇哥仍坚持自己的方向
- 累计亏掉一百多万

## 勇哥诊断

- 主题餐厅噱头大于实质
- 重装修等于把启动资金提前埋掉
- 烤鱼是流量生意，你把它变成了"看灯光生意"
- 听不进劝就只能继续亏

## 失败原因

1. 重资产病：装修一次性吃掉过多预算
2. 品类与玩法错配：烤鱼讲究效率，主题灯光拖累翻台
3. 复购薄弱：客人来一次拍照打卡，不会再来
4. 不听劝：多次连麦未做实质调整

## 给 skill 的启示

- 用户提到"主题 / 沉浸式 / 概念"+ 重装修预算 → 立即警告
- 主题餐厅默认算高风险品类（参见 corpus/05）
- 当用户已多次反馈仍拒绝调整 → 明确告知"下一步责任由你自己承担"
