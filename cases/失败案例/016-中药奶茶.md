# 案例 016：中药奶茶 · 三个月赔光 20 万

> 一句话摘要：把养生和奶茶强行嫁接，复购为零。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 016 |
| 当事人 | 一位创业者 |
| 时间 | 2024–2025 |
| 品类 | 概念奶茶（中药 / 养生） |
| 投资金额 | 20 万 |
| 经营时长 | 3 个月 |
| 最终结局 | 资金耗尽闭店 |

## 起因

加盟方主打"年轻人养生需求"，号称"00 后开始喝中药奶茶"。

## 经过

- 客单价高于普通奶茶
- 中药味导致接受度极低
- 第一次试新者多，复购几乎为零
- 3 个月内现金流崩溃

## 勇哥诊断

- 概念噱头 ≠ 长期需求
- 真有人养生不会用奶茶解决
- 中药 + 奶茶 = 两边的客户都不要你
- 复购才是餐饮命根，新奇感 30 天就没了

## 失败原因

1. 概念错配：养生客群和奶茶客群是两群人
2. 口味门槛：中药味让大众接受度极低
3. 复购支撑不足
4. 巨头价格战压力（蜜雪 / 古茗 / 茶百道）

## 给 skill 的启示

- 用户提到「中药 / 养生 / 国风健康」+ 高频快消品类 → 警惕概念噱头
- 评估复购率比评估首次新鲜感更关键
- "好味道才能引发复购，流量从来不是救命稻草"

## 出处

- 网易、搜狐元旦盘点
- 抖音「勇哥餐饮创业说」连麦切片
