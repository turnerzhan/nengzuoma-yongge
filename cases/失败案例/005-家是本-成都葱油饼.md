# 案例 005：家是本 · 成都葱油肉饼

> 一句话摘要：门头满是文字，"巨大历史机遇"比"葱油肉饼"还大。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 005 |
| 当事人 | 朱老板 |
| 时间 | 2026 年初 |
| 地点 | 成都龙泉驿区 |
| 品类 | 葱油肉饼小吃 |
| 投资金额 | 较小（具体未披露） |
| 经营时长 | 走红前已开 |
| 最终结局 | 因连麦走红，营业额反弹 |

## 起因

朱老板做的是普通葱油肉饼小吃，本身并无问题，但门头与营销极度抽象：

- 自称"首次提出中华传统文化三层经典基本认知共识理念"
- 门头上"巨大历史机遇"几个字比品类名还大
- 甚至有 AI 生成的主题曲

## 经过

- 经营平稳但门头风格让路人困惑
- 上勇哥连麦后被全网围观
- 网友用"诺贝尔一直讲"、"民间企业家请巴菲特吃午饭"等梗形容
- 走红后反而成为打卡圣地，营业额大幅提升

## 勇哥诊断

- 你这门头满是文字，"巨大历史机遇"比"葱油肉饼"还大
- 但是 —— 选品 OK，葱油饼是刚需小吃；房租也低
- 比起破产四件套那帮人，你这其实强多了
- 把门头清晰一点，别再让人猜你在卖啥

## 关键观察

这是少数 **"风格抽象 + 商业模型却合理"** 的反例：
- 葱油肉饼属于刚需小吃，复购可期
- 投入低，房租低，回本压力小
- 真正的问题只是门头识别度差

## 给 skill 的启示

- 不要因为"概念抽象"就一棍子打死
- 区分"概念抽象 + 模型也烂"（破产四件套）vs "概念抽象 + 模型其实 OK"（家是本）
- 门头识别度是 12 项观察清单中可以独立打分的项目
