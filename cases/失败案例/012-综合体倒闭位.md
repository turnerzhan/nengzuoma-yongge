# 案例 012：综合体倒闭位

> 一句话摘要：背靠盒马 + KTV 的"黄金位"，日销 4 单。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 012 |
| 当事人 | 一位中年大叔（带 14 岁女儿看店） |
| 时间 | 2025 年 |
| 地点 | 某商住综合体 |
| 品类 | 餐饮（普通快餐类） |
| 投资金额 | 约 20 万 |
| 最终结局 | 日销极少，孩子被叫来看店 |

## 起因

中介推介"背靠盒马 + 旁边 KTV"的位置，给当事人感觉是核心商圈口岸。

## 经过

- 实际位置在综合体某动线死角
- 盒马客流并不会自然导向该位置
- KTV 属于晚间业态，对快餐无加成
- 日销仅 4 单
- 当事人开始让 14 岁女儿在店里看店
- 连麦时被勇哥追问到细节，画面冲击力很强

## 勇哥诊断

- 商住综合体看着高端实则无客流
- "背靠 X 大牌"和"被 X 大牌带流量"完全是两回事
- 你看到的盒马客流 / KTV 客流 都不会进你这家店
- 让 14 岁孩子看店 = 你现在已经撑不住了
- 关，越早越好

## 失败原因

1. 错把"邻近"当"借势"：邻近不等于客流共享
2. 综合体陷阱：动线死角 / 业态不匹配
3. 中介信息不对称
4. 心理错觉："这么大的盒马旁边肯定不会差"

## 给 skill 的启示

- 用户提到「综合体 / 旁边是 X 大牌」→ 必须追问"动线"和"客流方向"
- 商住综合体在街景清单中默认红灯（参见 skill/街景观察清单.md）
- 当用户提到家庭成员（特别是孩子）参与经营 → 提示心理与法律风险
