# 案例 017：深圳中药馒头 · 及时止损

> 一句话摘要：罕见的"听劝"案例，亏损前止住。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 017 |
| 当事人 | 深圳一位创业者 |
| 时间 | 2025 年 |
| 地点 | 深圳 |
| 品类 | 中药馒头（概念主食） |
| 最终结局 | 在大额投入前止损 |

## 起因

被加盟方"养生主食 + 健康早餐"概念打动，进入到考察阶段。

## 经过

- 当事人在交付主要加盟款前找到勇哥
- 勇哥连麦中点出："中药馒头 = 早餐 + 概念噱头"，复购无支撑
- 当事人放弃签约，避免数十万损失

## 勇哥诊断

- 早餐是顺路生意，靠通勤路径
- 中药 + 馒头不是大众早餐
- 概念能驱动一次尝试，不能撑三年生意
- 这次你听劝，是省下了未来三年的眼泪

## 关键意义

这是案例库中**少数"在重大投入前回头"的案例**，价值在于：

1. 证明听劝是可以发生的
2. 证明 skill 类工具的真实价值（在掏钱之前给出风险提示）
3. 给所有潜在用户一个心理锚点："你也可以是这个案例"

## 给 skill 的启示

- 用户描述"还在考察 / 还没签合同 / 还在犹豫"时 → 这是 skill 最有价值的窗口期
- 必须直接说"在掏钱之前听这一句"
- 引用本案例做正向激励："深圳那哥们儿听劝省了几十万，你也能"
