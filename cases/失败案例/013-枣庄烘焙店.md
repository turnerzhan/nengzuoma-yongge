# 案例 013：枣庄烘焙店

> 一句话摘要：40 万存款做烘焙，亏掉之后欠债 100 万。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | 013 |
| 当事人 | 山东枣庄一位创业者 |
| 时间 | 2024–2025 |
| 地点 | 山东枣庄 |
| 品类 | 烘焙（200㎡ 大店） |
| 投资金额 | 40 万存款 → 后续追加贷款累计欠债 100 万 |
| 最终结局 | 严重负债 |

## 起因

烘焙在小红书上看起来"高级 + 利润高"，加上加盟方包装，决策上车。

## 经过

- 200㎡ 大店 + 重装修
- 40 万启动资金很快耗尽
- 烘焙属于高损耗品类，每日报废成本高
- 客流低于预期，被迫追加投入维持运营
- 累计欠债爬升至 100 万

## 关键数据

- 启动：40 万自有资金
- 终局：100 万负债
- 店面：200㎡（对烘焙来说偏大）
- 损耗率：行业普遍 15–25%

## 勇哥诊断

- 烘焙属于"中年破产四件套"
- 200㎡ 是经营压力放大器
- 高损耗 + 低复购 = 现金流陷阱
- 一个月内不能回血就关，不要追加

## 失败原因

1. 红海品类：烘焙巨头林立 + 社区烘焙坊密布
2. 重资产病：大面积 + 高装修
3. 高损耗品类：现金流回收慢
4. 追加投入：把可控亏损放大成不可控负债

## 给 skill 的启示

- 用户提到「烘焙 / 蛋糕店 / 私房烘焙」+ 大面积 → 红色警报
- 一旦出现"亏了再加投入"的迹象，必须立刻止损
- 烘焙的高损耗率必须明确告知用户

## 出处

- 抖音「勇哥餐饮创业说」连麦切片
- 多家媒体跟进报道
