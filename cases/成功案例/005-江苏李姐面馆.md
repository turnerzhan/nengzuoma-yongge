# 成功案例 005：江苏李姐面馆

> 一句话摘要：面条 + 卤蛋 + 微信群，把一家普通面馆做出复购。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | S005 |
| 当事人 | 江苏李姐 |
| 时间 | 2024–2025 |
| 地点 | 江苏 |
| 品类 | 面馆 |
| 最终结局 | 整改后稳定盈利 |

## 起因

李姐自己开面馆已经一段时间，能保本但增长乏力。咨询勇哥时数据基本健康，但缺乏增长抓手。

## 经过

- 勇哥诊断：保本达标，可以做差异化优化
- 调整动作：
- 第二碗免费续面
- 加微信送卤蛋
- 微信群里发明日特价 / 限量菜单
- 数月后复购率显著提升

## 勇哥诊断

- 你这位置和成本没问题，缺的是把熟客绑住的钩子
- 续面 + 卤蛋是低成本钩子，复购效果立竿见影
- 微信群是 0 成本会员体系
- 别再花钱搞装修了

## 成功原因

1. 基础健康：保本达标
2. 差异化是钩子而非概念：续面 / 卤蛋直接刺激复购
3. 私域低成本：微信群拉住熟客
4. 老板执行力强，调整后立刻见效

## 给 skill 的启示

- 用户已开店且保本达标 → 推送差异化复购钩子方案
- 不要默认用户都是亏损户，有一类是"健康但缺增长"
- 私域钩子（微信群）是低成本高回报的标准动作
