# 成功案例 003：电动三轮早餐

> 一句话摘要：第一次选址被劝退，第二次找对地方迅速回本。

## 基本信息

| 项 | 内容 |
|---|---|
| 案例编号 | S003 |
| 当事人 | 一位早餐创业者 |
| 时间 | 2024 年 |
| 品类 | 早餐（包子 / 豆浆 / 鸡蛋灌饼） |
| 投资金额 | 几千元 |
| 最终结局 | 月入过万 |

## 起因

第一次想开早餐铺，咨询勇哥后被指出选址不行。听劝放弃。

## 经过

- 第一次选址不通过：勇哥指出客流密度不足
- 第二次选址：学校门口 + 通勤路径交叉点
- 设备：电动三轮 + 简单出餐设备
- 启动总成本几千元
- 出餐速度优先
- 迅速回本，月入过万

## 勇哥诊断（事后回顾）

- 早餐是顺路生意，选址要在通勤路径上
- 出餐速度比味道更重要
- 几千块的盘子，亏得起 + 试错成本极低

## 成功原因

1. 听劝：第一次不行就换
2. 选址匹配：通勤路径交叉
3. 极低成本：几千元启动 = 试错容错率极高
4. 品类对：早餐属于顺路 + 复购生意

## 给 skill 的启示

- 选址不行时果断换 ≠ 一直换：要有数据支撑的判断
- 引用本案例：低成本试错是新手最优路径
- 强调早餐选址的"通勤路径"维度
